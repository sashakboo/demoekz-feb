# Портал клининговых услуг «Мой Не Сам»

Веб-приложение для формирования заявок на уборку жилых и производственных помещений.

## Технологии

- Frontend: React, Material UI
- Backend: Node.js, Express
- Database: PostgreSQL

## Установка и запуск

1. Убедитесь, что у вас установлены Node.js и npm
2. Установите зависимости для всего проекта:
```bash
npm run install-all
```

3. Создайте файл `.env` в директории `backend` и укажите параметры подключения к базе данных:
```env
DB_USER=postgres
DB_HOST=localhost
DB_NAME=cleaning_service
DB_PASSWORD=postgres
DB_PORT=5432
PORT=5000
```

4. Запустите приложение (backend и frontend одновременно):
```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Альтернативный запуск

Если вы хотите запустить backend и frontend отдельно:

### Backend
1. Перейдите в директорию backend:
```bash
cd backend
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите сервер:
```bash
npm run dev
```

Backend будет доступен по адресу http://localhost:5000

### Frontend
1. Перейдите в директорию frontend:
```bash
cd frontend
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение:
```bash
npm start
```

Frontend будет доступен по адресу http://localhost:3000

## Функционал

### Общие страницы:
- Регистрация пользователя
- Авторизация пользователя

### Для обычных пользователей:
- Просмотр истории заявок
- Создание новых заявок

### Для администратора:
- Просмотр всех заявок
- Изменение статуса заявок (новая, в работе, выполнено, отменено)
- Указание причины отмены заявки
