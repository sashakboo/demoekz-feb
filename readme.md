Запуск (все команды в  текущей директории):

восстановить npm пакеты:
```
npm run install-all
```


Запустить оба сервиса одновременно:
```
npm run dev
```


По умолчанию приложение подключается к облачной субд postgres. БД уже настроена и заполнена тестовыми данными.


Также, можно настроить свое подключение к БД в файле /backend/.env. Например:

```bash
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=11111
DB_NAME=demoekz
PORT=5000
```

При запуске в БД будет создан админ сайта:
```
логин: adminka
пароль: cleanservic
```
